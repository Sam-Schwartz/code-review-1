Sam's Epicodus Projects

I'm taking an introductory programming class at Epicodus in order to learn foundational programming skills and concepts.

Our entire curriculum is detailed online, but briefly, I'm learning to write in

  • HTML and CSS: The presentation and styling languages of the web, and
  • JavaScript: The programming language of the web.

I'm learning to augment the impact of basic CSS and JavaScript with

  • Bootstrap: A framework for easily creating good-looking websites, and
  • jQuery: A JavaScript library for making interactive web pages.

I'm also learning about how to initiate, track, and document my project work by using

  • Command Line: The tool programmers use to interact with their computers,
  • Git: A tool programmers use to track their code, and
  • Markdown: A simple language for documents.

Epicodus follows a pair-programming model. In each class, we work through a daily lesson with a partner to understand and practice new material. Then we create a project to showcase our newly aquired programming skills. By working in pairs, we hone the ability to speak clearly about programming concepts, identify points of confusion, and articulate problem-solving strategies. Our project work reinforces conceptual learning by requiring immediate application of new material. It's a great model, and I'm learning a lot!
